# Global Superstore Sales Performance Dashboard

## Project Overview

This project analyzes 51,290 global retail transactions from the Global Superstore dataset using Python, SQL, SQLite, and Power BI.

The objective was to identify key business insights related to sales performance, profitability, customer behavior, product performance, and regional trends.

## Tools & Technologies

* Python (Pandas, Matplotlib)
* SQL
* SQLite
* Power BI
* GitHub

## Key Performance Indicators (KPIs)

* Total Sales: $12.64M
* Total Profit: $1.47M
* Total Quantity Sold: 178K
* Profit Margin: 11.61%
* Total Customers: 17K

## Dashboard Features

* Sales by Category
* Profit by Category
* Top 10 Products by Sales
* Sales by Region
* Monthly Sales Trend
* Interactive Filters (Category, Region, Segment, Year)

## Key Insights

* Technology generated the highest sales ($4.74M)
* Technology generated the highest profit ($0.66M)
* Western Europe was the top-performing region
* Sales peaked during Q4 across multiple years
* Strong sales growth trend observed from 2012 to 2015
